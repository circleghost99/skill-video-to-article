# Video-to-Article 開發日誌

本目錄記錄 v2a Skill 的所有迭代歷史。

## 格式規範

每個版本一個檔案，命名：`YYYY-MM.md`（按月份歸檔）。

### 條目格式

```markdown
## [序號]. 標題（日期）

**問題**：一句話描述問題
**根因**：為什麼會發生
**修復**：改了什麼
**檔案**：涉及哪些檔案
**驗證**：怎麼確認修好了
**教訓**：（可選）避免再犯的經驗
```

### 分類標籤

在標題後可加分類：

- `[Gemini]` — 提示詞工程
- `[FFmpeg]` — 素材擷取
- `[Notion]` — 上傳/發布流程
- `[品質]` — 視覺品質控制
- `[架構]` — Skill 結構/合規
- `[DX]` — 開發體驗/流程優化
