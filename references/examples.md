# 觸發範例與邊界

## 什麼情況該觸發

當需求是把 **單支有字幕的 YouTube 影片** 整理成文章時，使用這個 skill。

常見例子：
- 「把這支 YouTube 影片整理成一篇可發的長文」
- 「幫我把這支有字幕的影片寫成解讀文，不要只是摘要」
- 「把這支影片整理成 markdown 文章草稿」
- 「把這支影片的內容重組成深度筆記」
- 「不要照時間軸抄，幫我寫成讀者看得懂的文章」

## 什麼情況不該觸發

以下情境不應使用這個 skill：
- 使用者只要短摘要
- 使用者只要逐字稿整理
- 使用者只要摘錄金句
- 使用者只要純翻譯
- 使用者只要做影片下載或單純截圖
- 使用者要整合多支影片（v1 先不處理）
- 影片沒有可用字幕，且使用者也沒有提供 transcript

## v1 預期結果

這個 skill 的交付物應該是以下其中一種：
- 可閱讀的單篇長文草稿
- 結構化深度筆記
- 可同步發佈的 markdown 文章
- 若字幕品質有限，則是有明確 caveat 的短篇解讀文

重點不是「拿到字幕」，而是「把影片變成真正可讀的文章」。
